03TENBAGGER LAB · 정밀 조건 발굴
FMP 데이터 · 성장주 스크리닝
Tenbagger Lab 10배 주 후보 발굴

아래 다섯 가지 텐베거 발현 조건을 조합해, 미국 상장사 중 10배 성장 잠재 후보를 걸러냅니다.

TENBAGGER CONDITIONS — 발현 조건 5
① 아직 작은 시가총액

$5B→$50B는 가능해도 $500B→$5T는 어렵습니다. 출발점이 작아야 10배가 산수로 성립합니다.

② 매출의 빠른 성장

텐베거의 연료. 이익보다 먼저 오는 연 15~20%+ 매출 고성장이 장기 복리를 만듭니다.

③ 높은 매출총이익률

마진은 가격결정력·해자의 흔적. 성장이 이익으로 전환될 때의 폭발력을 결정합니다.

④ 성장 대비 저평가

PSR은 적자 성장주도 재는 자. 낮을수록 시장이 아직 못 알아본 기업일 확률이 높습니다.

⑤ 고점 대비 눌림목

좋은 기업을 조정 구간에서. 진입가가 최종 수익률의 절반을 결정합니다.

숫자 칸을 비워두면 그 조건은 무시 FMP 무료 키 · 하루 250회 · 카드 등록 불필요
각 입력 조건의 상세 근거 — 텐베거 관점 (펼치기/접기)
시총 상한

10배의 산수 — 피터 린치의 텐베거 대부분이 중소형에서 나왔습니다. 기본값 $50B 이하.

매출 CAGR

최근 3년 연평균 매출 성장률. 15~20% 이상이 텐베거의 통과선입니다.

연속 증가

일회성 특수가 아닌 추세적 성장인지 — 꾸준함이 질적 증거입니다.

매출총이익률

SW·플랫폼 60%+, 제조업 30%+면 양호. 마진이 클수록 이익 전환 폭발력이 큽니다.

PSR

시총 ÷ 연매출. 성장률 대비 낮을수록 저평가. 통상 5 이하에서 기회를 찾습니다.

PER · 흑자

이익 나는 기업의 과열 점검·보수 필터. 초기 텐베거는 적자가 흔하니 상황 따라 비워두세요.

고점대비

52주 최고가 대비 하락률. 단, 하락엔 이유가 있을 수 있으니 개별 분석은 필수입니다.

검증 종목 수

종목당 API 2회 소모. 무료 250회/일 예산 안에서 조절하세요.

1검색 대상 — 어디서 찾을까
FMP 무료 API 키 발급 방법 — 처음이라면 여기부터 (2분 소요)
01 · 접속

site.financialmodelingprep.com 에 접속해 우측 상단 Sign Up을 누릅니다.

02 · 가입

Sign up with Google(구글 계정 가입)이 가장 빠릅니다. 무료 플랜이면 충분합니다.

03 · 대시보드

로그인 후 상단 메뉴의 Dashboard로 이동합니다.

04 · 키 복사

화면의 API Key 항목에 보이는 긴 영문·숫자 문자열이 내 키입니다. 복사하세요. (타인에게 공유 금지)

05 · 붙여넣기

아래 ① 검색 대상의 "FMP API 키" 칸에 붙여넣으면 이 브라우저에 자동 저장됩니다.

06 · 확인

옆의 키 진단 버튼으로 내 키가 정상 작동하는지 즉시 확인할 수 있습니다.

2성장 조건 — 얼마나 빠르고 꾸준한가
3밸류에이션 · 타이밍 — 비싸지 않은가, 지금인가
4실행
1차 후보 = 내장 유니버스(미국 주요 성장·대표주 ~500종목) + 직접 추가 티커. 무료 플랜이 스크리너 API를 막아(402) 우회하는 방식입니다.
랜덤 샘플링 — 실행할 때마다 후보를 새로 뽑으므로, 여러 번 돌리면 다른 종목이 나옵니다.
API 예산 — 정밀검증은 종목당 2회 호출(30종목 ≈ 60회 / 무료 250회·일). 키는 이 브라우저에만 저장됩니다.

HOW IT WORKS — 운영 방식
◆화면(코드)과 데이터는 따로 삽니다. 홈페이지의 디자인·기능은 GitHub Pages에, 내가 입력한 기록은 Google Firebase에 저장됩니다. 덕분에 사이트가 업데이트되거나 디자인이 바뀌어도 그동안 쌓아온 기록은 전혀 사라지지 않습니다.
◆저장 버튼이 없습니다. 무언가를 입력하거나 수정하면 약 1.5초 뒤 자동으로 클라우드에 반영됩니다. 따로 "저장"을 누를 필요 없이, 쓰는 순간 곧바로 보관됩니다.
◆보안은 서버가 강제합니다. 계정별 접근 제한은 Firebase 보안 규칙이 서버 차원에서 처리하므로, 설령 이 사이트의 코드가 공개되더라도 다른 사람이 내 데이터에 접근할 수 있는 통로는 열리지 않습니다.
HOW TO USE — 사용 방법
1로그인하기. 우측 상단 LOGIN → 구글 계정 선택. 처음 로그인하는 것이라면, 지금까지 이 기기에만 있던 기록이 자동으로 내 클라우드로 옮겨집니다 (기존 기록이 사라지지 않으니 안심하세요).
2평소처럼 쓰기. 이후에는 신경 쓸 것 없이 그냥 쓰면 됩니다. 지식금고(BOOKS·LEXICON·ECONOMICS·MEDIA), 자산(ASSETS), 보유종목(HOLDINGS), 텐베거 워치리스트·메모, 최근 검색, API 키까지 모두 자동으로 동기화됩니다.
3다른 기기에서 이어서. 집·회사·휴대폰 등 어디서든 같은 구글 계정으로 로그인하면 클라우드의 기록이 그대로 복원되어 이어서 작업할 수 있습니다.
4로그아웃 & 백업. 우측 상단 버튼(로그인하면 계정 이름으로 바뀜)을 다시 누르면 로그아웃됩니다 — 이 기기의 기록은 남고 자동 저장만 꺼집니다. 만약을 대비해 위쪽 BACKUP DATA 버튼으로 주기적으로 JSON 파일 백업도 받아두면 더욱 안전합니다.
